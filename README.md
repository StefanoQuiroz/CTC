# Instrucciones

```sh
1. Dentro de la terminal posicionarse en una carpeta.
2. git clone https://github.com/StefanoQuiroz/CTC.git
3. npm i (para descargar todas las dependencias empleadas).
4. npm run start (para ejecutar).
```
